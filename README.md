# Приложение для решения уравнений методом Крамера

**SLAE_KramerMethodAPP** — это учебный проект, разработанный для:
- Освоения применеия рекурсии для решения задач на практике
- Знакомства с потоковым чтением текстовых файлов, при помощи объекта класса ``StreamReader``
- Обучения грамотно и **понятно** писать код

---

## Функции
1. Вывод *близкой* к алгебраической записи уравнения
2. Рекурсивный расчёт определителя матрицы (**Метод:** ``Determinant(int[,] mas)``)
3. Расчёт всех **x** для данного уравнения

* 1 и 3 пункты расположены в методе (``Main(string[] args)``)

---
